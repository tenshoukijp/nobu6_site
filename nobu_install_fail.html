天翔記 with PK for Windows95の インストーラー

概要

このインストーラーは、「天翔記 with PK for Windows95」を適切にインストールするためのインストーラーとなっています。
Windows11/10/8.1/8.0/7/Vista/XP などに対応しています。

ダウンロード

更新日 %(year)04d/%(mon)02d/%(mday)02d
Ten95PK_installer.zip

動作環境

.NET FrameWork 4.0 ～ 4.8

使い方

  1. 上記プログラムを、ダウンロード。zipを解凍し、Ten95PK_installer.exe を、
    適当な場所(クリックしやすい場所)に移動させます。(PCのデスクトップなどでもOK）

  2. 次に、PCのドライブに「天翔記 with PK for Windows95」のCD-ROMを挿入します。
    (仮想ドライブでも良い)
  3. この状態で、Ten95PK_installer.exe をクリックすると、インストールが始まります。
  4. 「C:\_tenshouki95」フォルダ等にインストールされます。
    (ちゃんと存在する固有ドライブにインストールされます。)

  5. インストールが完了した後、
    この「C:\_Tenshouki95」のフォルダを自由に好きなフォルダへと移動させたり、
    フォルダ名の変更等しても大丈夫です。
  6. Ten95PK_installer.exe は役割を終えましたので、削除してください。

補足説明

以下の４つのことが行われます

  1. 天翔記 with PK for Windows95のCDからインストールされます。
  2. その際、天翔記付属のsetupは使われませんので、それに伴うトラブルが回避されます。
  3. 天翔記をフルカラー画面のまま起動できるようにファイルを追加し、適切に設定等対処がなされます。
  4. 天翔記のバージョンを「定番シリーズ」と同じ状態へと更新します。
