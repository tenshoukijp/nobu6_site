On_終了時()

void カスタム::On_終了時() {
}

タイミング

  • 見た目的には、天翔記が終了して、ウィンドウフレームが消えた後に呼び出される。
    タイミング的には、TSModがDLLが解放され、全ての役割を終える直前のタイミングである。

一般的用途

  • On_起動時()などにおいて、
    独自に「動的メモリ確保」や独自に「LoadLibrary」をしている場合は、
    このメソッドで解放することになるだろう。

例①

int *p;
void カスタム::On_起動時() {
  p = new int;
}

void カスタム::On_終了時() {
  delete p;
}

例②

HMODULE pDLL;
void カスタム::On_起動時() {
  pDLL = LoadLibrary("MyCustomMode.dll");
}

void カスタム::On_終了時() {
  FreeLibrary(pDLL);
}
