string カスタム::On_特殊兵態名表示直前(int 武将番号)

string カスタム::On_特殊兵態名表示直前(int 武将番号) {
  return "";
}

タイミング

  • このメソッドは武将の特殊兵隊を表示するタイミングで呼び出される。
  • 具体的には、様々な「武将リスト詳細画面」等と「武将詳細画面」である。

一般的用途

  • 自作のオリジナルな武将や、特に思い入れの強い武将の場合、 「新しい特殊兵隊名を付けたい」という目的で利用する。

例①

string カスタム::On_特殊兵態名表示直前(int 武将番号) {

  int iBushouID = 武将番号-1; // 武将番号→武将番号【配列用】に

  if ( 0 <= iBushouID && iBushouID < 最大数::武将情報::配列数 ) {
    if ( p武将戸籍情報[iBushouID].顔番号 == 顔番号::武田晴信 ) { // 武田信玄の顔＆列伝番号
      return "風林火山";
    }
  }

  // 変更しない場合""
  return "";
}
