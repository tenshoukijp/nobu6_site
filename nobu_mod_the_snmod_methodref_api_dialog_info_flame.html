ダイアログ - 情報フレームダイアログ

特にボタンや選択肢がないフレーム表示のダイアログを表示するAPIです。

大小関わらず、自作の独自イベントなどで利用することになるでしょう。

void 情報フレームダイアログ表示(string ダイアログ文字列)
void カスタム::On_相場変更直前() {
	情報フレームダイアログ表示("そうですわ");
}

void カスタム::On_残りターン変更《戦争画面》(int 残りターン数) {
    if ( 残りターン数 < 30 ) {
	    情報フレームダイアログ表示("あいう");
    }
}
