C++/CLI 必須ツール

  • 概要

    ここでは、C++/CLI でプログラムを記述・学習する上において、必須のオーサリングツールを紹介します。

  • C++/CLI .NET GUI 編集ツール

    • Visual Studio 2008

      今となっては相当古くなってしまったVisual Studio 2008ですが、
      実は、C++/CLIの.NET FrameWork系のオーサリング機能が最も充実していたのが、
      このVisual Studio 2008 Standard(以上)です。
      (2005でも2010以降でもなく、2008のみが該当します)

      このバージョンのVisual Studioは、C++/CLIでWindows Form系のアプリや設定をWYSIWYGに編集可能であり、
      オーサリングされたソースを見て、非常に効率良く学習を進めることが出来ます。
