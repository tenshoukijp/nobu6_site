string カスタム::On_家宝種類名表示直前(int 家宝番号, string 家宝名)

string カスタム::On_家宝種類名表示直前(int 家宝番号, string 家宝名) {
  return "";
}

タイミング

  • このメソッドは対象の家宝の種類名(刀剣や馬といったもの)を表示するタイミングで呼び出される。
  • 具体的には、あらゆる「家宝リスト画面」と「家宝詳細画面」である。

一般的用途

  • 自作のオリジナルな家宝など作った場合、既存の家宝種類で当てはまるものが無い場合など、 「新しい家宝種類の名前を付けたい」という目的で利用する。

例①

string カスタム::On_家宝種類名表示直前(int 家宝番号, string 家宝名) {

  if ( 家宝名 == "孫子" ) {
    return "軍学書";
  }

  else if ( Is_正規表現マッチ(家宝名, ".+弓$") ) { // 自作家宝名で、最後の文字が「弓」のものは全て
    return "弓";
  }

  // 変更しない場合""
  return "";
}
