シナリオメニュー型＆ カスタム::On_シナリオメニュー表示直前()

シナリオメニュー型＆ カスタム::On_シナリオメニュー表示直前() {
  return NULL;
}

タイミング

  • 見た目的にもタイミング的にも、６つのシナリオメニューが出る直前で呼び出される。
    又、「セーブデータ」の「ロード画面表示・セーブ画面表示」も、シナリオメニュー表示の一種とみなされるため、呼び出される。


一般的用途

  • ６つのメニューの文字列を書き換えるために利用される。

例①

シナリオメニュー型＆ カスタム::On_シナリオメニュー表示直前() {

  シナリオメニュー型 menu = {
    "  三人の養子 （１５０７年）",
    "",                             // 元のまま
    "  相剋の果て （１５６５年）",
    "信長包囲網･改（１５７１年）",
    "本能寺の変･改（１５８２年）",
    "関ヶ原前夜･改（１５９９年）"
  };

  return menu;

  // 変更しない場合NULL
  return NULL;
}
