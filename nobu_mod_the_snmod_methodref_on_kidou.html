On_起動時()

void カスタム::On_起動時() {
}

タイミング

  • 見た目的には、天翔記が起動して、ウィンドウフレームが描画される直前に呼び出される。
    タイミング的には、TSModが各種必要な設定や準備を全て終えた時に呼び出される。


一般的用途

  • 何か変数を初期化したり、外部ＤＬＬを呼び出すのに利用することが多いだろう。

例①

int myVariable = 0;
void カスタム::On_起動時() {
  myVariable = 1;
}

例②

void カスタム::On_起動時() {
  Set_効果音ＤＬＬ追加("SnMyKoukaon.dll");
}
